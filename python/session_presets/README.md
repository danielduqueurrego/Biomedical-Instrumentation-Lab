# Session Presets

Reusable JSON presets for fast lab setup in the GUI.

## Included presets
- `ecg.json`
- `emg.json`
- `pulse_ox.json`
- `blood_pressure.json`

## What is saved in a preset
- selected lab profile
- active signal names/types/ports
- acquisition class and rate defaults
- save filename/folder defaults
- plotting layout choices

## Typical classroom flow
1. Launch GUI
2. Click **Load Preset**
3. Confirm board and save folder
4. Compile/upload if needed
5. Start acquisition
