# Validation

Quick, repeatable checks for instructors and TAs before class.

## Fast workflow
1. Open the matching lab guide
2. Load preset in GUI
3. Compile/upload firmware
4. Run a short capture (30–60 s)
5. Save CSV and review packet quality

## Record these values
- target rate
- achieved rate
- test duration
- malformed/parse errors
- disconnect or serial issues
