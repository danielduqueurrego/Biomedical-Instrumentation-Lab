# Screenshot Placeholders

This folder is reserved for classroom and validation screenshots.

Nothing in this folder is required for the software to run. It exists so TAs and instructors have a consistent place to store images that may later be added to lab handouts or validation notes.

Suggested screenshot types:
- GUI setup before acquisition
- live plot during acquisition
- board wiring overview
- sensor placement or bench setup
- validation evidence for known-good runs

Suggested naming style:
- `emg_gui_setup.png`
- `ecg_live_plot.png`
- `pulse_ox_board_wiring.jpg`
- `blood_pressure_live_plot.png`
- `validation_emg_plot.png`

If a screenshot is referenced from a lab guide or checklist, place it here.
