# Screenshots

> Placeholder and storage area for screenshots used in setup guides, lab guides, and validation notes.

Use this folder when you want to replace the current placeholders with real classroom screenshots or add new images for instructors and TAs.

---

## Start Here

Nothing in this folder is required for the software to run. It exists so the docs always have a consistent place to point when a screenshot is needed.

Current placeholder assets:

- `student_gui_setup_placeholder.svg`
- `emg_live_plot_placeholder.svg`
- `pulse_ox_live_plot_placeholder.svg`

## Recommended Replacements

- replace `student_gui_setup_placeholder.svg` with `student_gui_setup.png`
